## 一、社区功能扩展

### 1.1 搜索功能
**优先级：高**

- [ ] **搜索界面设计**
  - 全局搜索框（顶部导航栏）
  - 搜索结果页面（分类展示：文章/问题/用户）
  - 高级筛选（按（时间、热度、标签过滤）
- [ ] **搜索功能实现**
  - 全文搜索（文章标题、内容、标签）
  - 问题搜索（标题、内容）
  - 用户搜索（用户名、简介）
  - 搜索历史记录
- [ ] **搜索体验优化**
  - 搜索建议/自动补全
  - 热门搜索词展示
  - 高亮关键词显示

### 1.2 个人主页信息扩展
**优先级：高（）

- [ ] **个人定位信息**
  - 添加地区/省份选择器
  - 添加经纬度坐标（可选）
  - 在个人资料中展示种植地点
- [ ] **我的植物信息**
  - 添加"我的植物"板块
  - 支持添加/编辑种植作物清单
  - 显示种植面积、种植时间
  - 作物生长阶段状态追踪

### 1.3 学习模块（新增）
**优先级：高**

- [ ] **学习课程设计**
  - 结构化课程分类（作物种植、土壤管理、病虫害防治、农业技术等）
  - 每门课程包含多个章节/课时
  - 支持图文、视频等多媒体内容
- [ ] **学习进度追踪**
  - 记录用户学习进度（已完成/进行中/未开始）
  - 学习时长统计
  - 课程完成率与成就系统
- [ ] **互动学习功能**
  - 课后小测验/练习题
  - 学习笔记功能
  - 学习讨论区（每门课程专属讨论）
- [ ] **推荐系统**
  - 根据用户兴趣和历史行为推荐课程
  - 热门课程排行榜
  - 学习路径推荐（新手→进阶→专家）

### 1.4 文章 AI 总结
**优先级：高 | 状态：已完成**

- [x] **AI 自动总结**
  - 文章发布后自动生成 200 字以内的摘要
  - 提取文章关键词/标签
  - 支持一键复制摘要用于分享
- [x] **用户交互优化**
  - 在文章列表页显示 AI 摘要（替代手动填写摘要）
  - 长文章顶部显示"AI 速览"折叠框
  - 支持一键"AI 帮我总结"按钮（针对历史文章）

### 1.5 问题 AI 解答
**优先级：高 | 状态：已完成**

- [x] **AI 优先回答**
  - 用户提问后，AI 自动生成参考答案
  - 参考答案标注"AI 生成"标识
  - 用户可选择采纳 AI 答案或等待人工回答
- [x] **问题智能分类**
  - AI 自动识别问题领域（如作物/土壤/设备）
  - 自动匹配相关专家或相似问题
  - 问题质量评估（描述是否清晰、是否需要补充信息）
- [x] **回答质量增强**
  - AI 辅助人工回答（提供回答模板/建议）
  - AI 检测回答质量并给出改进建议
  - 最佳答案自动推荐

## 三、AI 对话功能升级

### 3.1 多模态支持（图片处理）
**优先级：高**

- [ ] **图片上传与处理**
  - 前端支持图片拖拽/粘贴/选择上传
  - 图片压缩与格式优化（WebP/JPEG/PNG）
  - 图片大小限制（如最大 5MB）
- [ ] **视觉理解能力**
  - 升级到支持视觉的多模态模型（如 GPT-4V、DeepSeek-VL 等）
  - 图片识别场景：
    - 作物病虫害诊断（上传叶片照片识别病害）
    - 土壤状况分析（上传土壤照片判断质地/湿度）
    - 作物生长状态评估
    - 农业设施/设备/识别
- [ ] **对话交互优化**
  - 对话中支持图文混排显示
  - 图片预览与放大功能
  - AI 可以在回复中生成图片（如作物生长示意图）
- [ ] **技术实现要点**
  - 修改 `llmService.js` 支持多模态消息格式
  - 更新消息表结构（添加图片 URL 字段）
  - 前端消息组件支持图片渲染
  - 图片存储方案（本地存储或对象存储 OSS）
