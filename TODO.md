# GrowPal 开发 TODO 清单

> 最后更新：2026-04-12

---

## 一、社区功能扩展

### 1.1 学习模块（新增）
**优先级：高**

在社区现有文章和问题功能基础上，新增学习模块，支持：

- [ ] **学习课程设计**
  - 结构化课程分类（作物种植、土壤管理、病虫害防治、农业技术等）
  - 每门课程包含多个章节/课时
  - 支持图文、视频等多媒体内容
- [ ] **学习进度追踪**
  - 记录用户学习进度（已完成/进行中/未开始）
  - 学习时长统计
  - 课程完成率与成就系统
- [ ] **互动学习功能**
  - 课后小测验/练习题
  - 学习笔记功能
  - 学习讨论区（每门课程专属讨论）
- [ ] **推荐系统**
  - 根据用户兴趣和历史行为推荐课程
  - 热门课程排行榜
  - 学习路径推荐（新手→进阶→专家）

### 1.2 文章 AI 总结
**优先级：高**

- [ ] **AI 自动总结**
  - 文章发布后自动生成 200 字以内的摘要
  - 提取文章关键词/标签
  - 支持一键复制摘要用于分享
- [ ] **用户交互优化**
  - 在文章列表页显示 AI 摘要（替代手动填写摘要）
  - 长文章顶部显示"AI 速览"折叠框
  - 支持一键"AI 帮我总结"按钮（针对历史文章）

### 1.3 问题 AI 解答
**优先级：高**

- [ ] **AI 优先回答**
  - 用户提问后，AI 自动生成参考答案
  - 参考答案标注"AI 生成"标识
  - 用户可选择采纳 AI 答案或等待人工回答
- [ ] **问题智能分类**
  - AI 自动识别问题领域（如作物/土壤/设备）
  - 自动匹配相关专家或相似问题
  - 问题质量评估（描述是否清晰、是否需要补充信息）
- [ ] **回答质量增强**
  - AI 辅助人工回答（提供回答模板/建议）
  - AI 检测回答质量并给出改进建议
  - 最佳答案自动推荐

## 二、AI 对话功能升级

### 2.1 多模态支持（图片处理）
**优先级：高**

- [ ] **图片上传与处理**
  - 前端支持图片拖拽/粘贴/选择上传
  - 图片压缩与格式优化（WebP/JPEG/PNG）
  - 图片大小限制（如最大 5MB）
- [ ] **视觉理解能力**
  - 升级到支持视觉的多模态模型（如 GPT-4V、DeepSeek-VL 等）
  - 图片识别场景：
    - 作物病虫害诊断（上传叶片照片识别病害）
    - 土壤状况分析（上传土壤照片判断质地/湿度）
    - 作物生长状态评估
    - 农业设施/设备识别
- [ ] **对话交互优化**
  - 对话中支持图文混排显示
  - 图片预览与放大功能
  - AI 可以在回复中生成图片（如作物生长示意图）
- [ ] **技术实现要点**
  - 修改 `llmService.js` 支持多模态消息格式
  - 更新消息表结构（添加图片 URL 字段）
  - 前端消息组件支持图片渲染
  - 图片存储方案（本地存储或对象存储 OSS）

## 三、其他推荐方向

### 3.1 个性化推荐系统
**优先级：中**

- [ ] **内容推荐**
  - 基于用户兴趣的社区内容推荐
  - 热门文章/问题/课程推送
  - 相似用户推荐（关注/粉丝系统）
- [ ] **AI 助手个性化**
  - 记忆用户偏好和历史对话
  - 根据用户作物类型定制建议
  - 用户档案增强（种植经验水平、所在地区、主要作物）

### 3.2 数据可视化与报告
**优先级：中**

- [ ] **农业数据仪表盘**
  - 用户种植记录统计（作物种类、面积、时间线）
  - 学习进度可视化图表
  - AI 对话使用统计
- [ ] **周期性报告**
  - 周报/月报生成（种植进度、学习成就、AI 使用情况）
  - 可导出 PDF 报告
  - 种植建议与提醒（基于季节和作物生长周期）

### 3.3 社区互动增强
**优先级：中**

- [ ] **社交功能**
  - 用户关注/粉丝系统
  - 点赞/收藏/评论功能完善
  - 用户等级与成就徽章系统
- [ ] **内容创作激励**
  - 创作者认证（农业专家/种植达人）
  - 优质内容打赏/积分系统
  - 内容质量评分与排名
- [ ] **活动与话题**
  - 周期性农业主题活动（如"春季种植挑战赛"）
  - 话题标签系统（#有机种植 #节水灌溉）
  - 专家在线答疑活动

### 3.4 移动端优化与 PWA
**优先级：中**

- [ ] **PWA 支持**
  - 离线访问支持（缓存文章/课程）
  - 添加到主屏幕支持
  - Push Notification（问题回答通知、学习提醒）
- [ ] **移动端体验优化**
  - 响应式设计优化
  - 手势操作支持（左滑删除、下拉刷新）
  - 拍照上传优化（田间照片快速分享）

### 3.5 AI 能力扩展
**优先级：中**

- [ ] **语音交互**
  - 语音输入转文字（适合田间使用场景）
  - AI 语音回复朗读
  - 语音命令支持
- [ ] **智能提醒与预警**
  - 基于天气/季节的种植提醒
  - 病虫害预警推送
  - 学习进度提醒
- [ ] **多模型支持**
  - 支持多个 LLM 提供商（OpenAI、Claude、文心一言等）
  - 模型自动降级与容错
  - 用户可选模型（高级功能）

### 3.6 国际化与本地化
**优先级：低**

- [ ] **多语言支持**
  - 中英文切换
  - AI 回复多语言支持
  - 社区内容多语言展示
- [ ] **区域适配**
  - 不同地区作物推荐数据库
  - 本地化农业知识库
  - 区域气候数据集成

## 四、技术债务与优化

- [ ] **分页与懒加载**

  - 虚拟滚动支持（前端长列表）
- [ ] **图片与静态资源**

- [ ] **安全加固**

- [ ] **监控与日志**
